# Project-ChatApp

---

### 주제

Flutter와 Firebase를 활용한 **Chat App**💬 개발

### 기술 스택

Flutter, Firebase

### 기능

- 사용자 인증 - 로그인
- 사용자 상태 설정
- 목록에서 사람 검색
- 실시간으로 메시지 주고 받기

### pages

- hello
- login
- people
- chats
- message
- settings
